This page is dedicated to the helping users use AxoNet. If the infomation you need isn't on this page, ask the community!

Creating pages

AxoNet runs through GitHub, at https://github.com/AxoWasTaken/AxoNet/. Assuming you know how GitHub works, create a file named index.html and place it in a folder with a unique and descriptive name that describes your site. Write in the HTML for your site in that file, commit, and bam! You've got an AxoNet site. Please make sure that you don't add, edit, or remove anything at the root of the repository (as it is reserved for official AxoNet pages). If you want to you can also add a styling with a .css file. Also make sure that all images and assets for your site are foldered nicely, don't want a repeat of the Unison website 🤣.

Indexing pages

If you need your site indexed, email AxoWasTaken@outlook.com and wait for a response. If you want access to the Google Search Console you can, just ask us. Due to the fact google are a bit lazy it might take a day or two for the site to be indexed.

Error 404

If you get an 'Error 404 page not found' try searching for a typo in the URL you entered. It could be that the page you tried to reach was moved, deleted, or it never existed at all. Its worth it to check the repository[↗] as all the code is there. There should be a search box, enter the name of the site you are trying to access and check what the results are.
