Creating pages

AxoNet runs through GitHub, at https://github.com/AxoWasTaken/AxoNet/. Assuming you know how GitHub works, create a file named index.html and place it in a folder with a unique and descriptive name that describes your site. Write in the HTML for your site in that file, commit, and bam! You've got an AxoNet site. Please make sure that you don't add, edit, or remove anything at the root of the repository (as it is reserved for official AxoNet pages)
